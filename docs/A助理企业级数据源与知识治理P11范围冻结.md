# A助理企业级数据源与知识治理 P11 范围冻结

## 阶段定位

P11 是 Real CloudReadonly Pilot 准入演练阶段，只在 AICopilot 内完成上线前闸门、配置包、审批链、fake contract、拒绝样例和审计证据闭环。

P11 不表示已经进入生产 Pilot，也不表示已经启用真实 Cloud 生产数据读取。

## 固定边界

- 只修改 `AICopilot`。
- 不修改 `IIoT.CloudPlatform`。
- 不修改 `IIoT.EdgeClient`。
- 不启用 `CloudReadonly.Mode=Real`。
- 不启用 `CloudReadonly.Real.Enabled`。
- 不启用 `CloudReadonly.Real.AllowProductionRead`。
- 不启用 `CloudAiRead.Enabled`。
- 不让 Agent Runtime 读取真实 Cloud 生产数据。
- 不让 `query_cloud_data_readonly` 进入 Planner 可见工具目录。
- 不让 `query_cloud_data_readonly` 可被 Agent 执行。
- 不回显 API Key、token、连接串、密码或完整 payload。

## P11 允许内容

- 新增 `CloudReadonlyPilotReadiness` 配置段，默认 `Enabled=false`。
- 新增 Pilot readiness 配置包、gate 状态、审批演练和 fake contract 演练。
- 使用 fake production-like contract 验证 `devices`、`capacity_summary`、`device_logs`、`pass_station_records` 四类只读端点。
- 对 Recipe、Recipe version、写路径、未知 endpoint、超范围请求保持 `BlockedByPolicy`。
- 新增只读工具目录描述 `query_cloud_pilot_readiness_readonly`，但默认 disabled / hidden / non-executable。
- 前端展示 P11 Pilot 准入演练状态，并明确显示未启用生产读取、未接入真实生产数据。

## 完成口径

P11 完成后，系统只能证明：生产读取仍关闭，同时已经具备提交后续受控生产 Pilot 授权评审所需的配置、审批、契约、审计和风险证据。

后续如果要进入真实生产只读 Pilot，必须另行形成 P12 或专项授权计划。
