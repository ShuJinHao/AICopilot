# A助理企业级数据源与知识治理 P12 范围冻结

## 阶段边界

P12 只在 `AICopilot` 内实现固定模板、限时、审批式 Cloud 生产只读 Pilot。不得修改 `IIoT.CloudPlatform` 或 `IIoT.EdgeClient`。

## 默认关闭

- `CloudReadonlyProductionPilot.Enabled=false`
- `CloudReadonly.Mode=Disabled`
- `CloudReadonly.Real.Enabled=false`
- `CloudReadonly.Real.AllowProductionRead=false`
- `query_cloud_data_readonly` 保持 disabled / hidden / non-executable

## 允许范围

- 固定模板：设备清单、产能汇总、设备日志、过站记录、设备异常分析、产能交付分析
- 端点 allowlist：`devices`、`capacity_summary`、`device_logs`、`pass_station_records`
- 执行前置：P11 gate 通过、Pilot Window 生效、窗口审批完成、工具审批完成、端点在 allowlist 内

## 禁止范围

- 不开放自由目标生产查询
- 不读取 Recipe / Recipe version
- 不开放写路径
- 不开放 Cloud/Edge 联动
- 不回显 token、API Key、连接串、完整 payload 或敏感上下文
