# A助理企业级数据源与知识治理 P13 范围冻结

## 阶段边界

P13 只修改 `AICopilot`。本阶段不修改 `IIoT.CloudPlatform`，不修改 `IIoT.EdgeClient`，不做 Cloud/Edge 联动，不开放 Cloud 写入。

P13 是 Real CloudReadonly 受控自由目标生产只读 Pilot，不是全面生产上线。默认生产读取仍关闭，只有 P12/P11 gate 通过、Production Controlled Pilot Window 生效、自由目标 intent 映射合法、Tool Approval 完成、endpoint 在 allowlist 内时，P13 专用工具才可被临时执行。

## 默认关闭项

- `CloudReadonly.Mode=Disabled`
- `CloudReadonly.Real.Enabled=false`
- `CloudReadonly.Real.AllowProductionRead=false`
- `CloudReadonlyProductionControlledPilot.Enabled=false`
- `CloudReadonlyProductionControlledPilot.FreeGoalEnabled=false`
- `query_cloud_data_readonly` disabled / hidden / non-executable
- `query_cloud_production_controlled_readonly` disabled / hidden / non-executable

## 允许范围

P13 只允许把自由目标映射到以下只读端点：

- `devices`
- `capacity_summary`
- `device_logs`
- `pass_station_records`

所有查询结果和产物必须携带：

- `sourceMode=CloudReadonlyProductionControlledPilot`
- `isProductionData=true`
- `boundary=ProductionControlledPilot`
- `pilotWindowId`
- `intentId`
- `endpointCode`
- `queryHash/resultHash`
- `rowCount`
- `isTruncated`
- `approvalStatus`

## 禁止范围

- 不开放任意生产查询
- 不开放生产自由 SQL
- 不开放 Cloud 写接口
- 不开放 Recipe / Recipe version
- 不允许写路径、未知 endpoint、任意 payload 或任意服务器路径
- 不记录 token、API Key、连接串、完整 payload、完整敏感上下文
