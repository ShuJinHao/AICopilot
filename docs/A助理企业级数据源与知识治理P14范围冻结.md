# A助理企业级数据源与知识治理 P14 范围冻结

## 阶段定位

P14 是生产只读 Pilot 运营闸门与 GA 准入评估阶段。它只收口 P12 固定模板 Pilot 和 P13 受控自由目标 Pilot 的运营、停用、审计、风险和 P15 准入证据，不扩大生产读取范围。

## 修改边界

- 仅修改 `AICopilot`。
- 不修改 `IIoT.CloudPlatform`。
- 不修改 `IIoT.EdgeClient`。
- 不新增 Cloud 端点。
- 不开放 Cloud 写。
- 不开放 Recipe / Recipe version。
- 不开放 `query_cloud_data_readonly`。

## 默认安全状态

- `CloudReadonly.Mode=Disabled`。
- `CloudReadonly.Real.Enabled=false`。
- `AllowProductionRead=false`。
- P12/P13 Pilot 仍由各自 gate、window、approval 控制。
- `query_cloud_data_readonly` 继续 disabled / hidden / non-executable。

## P14 新增能力

- Production Pilot Operations Center。
- P12/P13 run ledger 汇总。
- runtime emergency stop。
- incident 台账。
- 脱敏运营指标。
- P15 readiness assessment。

## 明确非范围

- 不是全面生产上线。
- 不是 P15 GA。
- 不是自由 SQL 或任意生产查询。
- 不是 Cloud/Edge 联动。
- 不是长期持久化运营证据系统。
